Introducing ActionFPS

ActionFPS is a new match league for AssaultCube, following on from the original woop.ac.

Like duel.gg, it is fully open sourced on GitHub: github.com/ScalaWilliam/ActionFPS/ and comes with Open Data APIs which you can build on top of.

It comes with realtime game display, !inter functionality, achievements, clanwars.

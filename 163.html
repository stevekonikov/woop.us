Woop league has started!

The Woop League has started once again! The league is very simple: play AC or Sauer with anyone else on the table and when you're finished, add the result to the table. For more information, see http://woop.us/table (rules, how it works etc.) and the table itself is available at http://league.woop.us/.

If you're too lazy to read all the rules: you can play whichever game you want, whatever map you want, and how many times you want!

If you would like to add a game, Drakas will give you an access password.

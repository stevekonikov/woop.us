Customising some ActionCube options

I saw some people few days ago asking about automatic item chooser instead of having to press "B" and choose a weapon every time you launch ActionCube. Basically, edit your config/autoexec.cfg. Mine looks like this:

// this file gets executed if you start the game
// Shut off any music
musicvol 0
bind "C" "say :)"
bind "X" "say WATCH OUT! COMING BACK WITH THE FLAG!"
// Zoom with Left Alt
bind "LALT" "zoom 1; onrelease [ zoom 0 ]"
// Jump with right mouse
bind "MOUSE2" "jump"
// Increase the brightness/gamma
gamma 150
// Choose shotgun
item 2
// My best sensitivity
sensitivity 70
